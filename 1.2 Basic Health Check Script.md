# 📘 1.2 Basic Health Check Script

## 📋 Overview

**Purpose:** Monitor domain controller status, replication, and basic Active Directory health.

| Property      | Value                                                              |
| ------------- | ------------------------------------------------------------------ |
| 🎯 Purpose    | Monitor domain controller status, replication, and basic AD health |
| 📈 Difficulty | Easy                                                               |
| 💼 Use Case   | Daily health monitoring                                            |

---

# 📄 Script Code

```powershell
# L1.2: Basic Health Check Script
# ==================================

# GET AD DOMAIN INFO
function Get-ADDomainHealthStatus {
    param([string]$Domain = (Get-ADDomain).DNSRoot)

    Write-Host "=== AD Domain Health Report ===" -ForegroundColor Cyan
    Write-Host "Domain: $Domain`n"

    # Domain Controllers
    $DCs = Get-ADDomainController -Filter * -Server $Domain

    Write-Host "Domain Controllers ($($DCs.Count)):" -ForegroundColor Yellow

    foreach ($DC in $DCs) {
        Write-Host " • $($DC.Name) - $($DC.OperatingSystem)"
    }

    # Replication Status
    Write-Host "`nReplication Status:" -ForegroundColor Yellow

    $Replication = Get-ADReplicationStatus -EnumerationServer (Get-ADDomainController).HostName

    if ($Replication) {
        Write-Host " 3 Replication healthy"
    }
    else {
        Write-Host " 7 Replication issues detected"
    }

    # User & Computer Count
    $Users = (Get-ADUser -Filter *).Count
    $Computers = (Get-ADComputer -Filter *).Count

    Write-Host "`nDirectory Statistics:" -ForegroundColor Yellow
    Write-Host " Users: $Users"
    Write-Host " Computers: $Computers"
    Write-Host " Groups: $((Get-ADGroup -Filter *).Count)"
}

# CHECK DC CONNECTIVITY
function Test-DCConnectivity {
    param([string[]]$DomainControllers)

    Write-Host "`n=== DC Connectivity Test ===" -ForegroundColor Cyan

    foreach ($DC in $DomainControllers) {
        $Ping = Test-NetConnection -ComputerName $DC -WarningAction SilentlyContinue
        $Status = if ($Ping.PingSucceeded) { "3 Online" } else { "7 Offline" }

        Write-Host "$DC : $Status"
    }
}

# CHECK EVENT LOG FOR ERRORS
function Get-ADEventLogErrors {
    param([int]$LastHours = 24)

    Write-Host "`n=== Recent AD Errors ===" -ForegroundColor Cyan

    $StartTime = (Get-Date).AddHours(-$LastHours)

    Get-WinEvent -FilterHashtable @{
        LogName = 'Directory Service'
        Level = 2,3
        StartTime = $StartTime
    } -ErrorAction SilentlyContinue |
    Select-Object TimeCreated, Message |
    Format-Table -AutoSize -Wrap
}
```

# ⚙️ Functions Included

## 🏢 1. Get AD Domain Health Status

**Function:** `Get-ADDomainHealthStatus`

Generates an overall Active Directory health report including:

* Domain information
* Domain Controllers
* Replication status
* User count
* Computer count
* Group count

### Parameter

| Parameter | Description                                                 |
| --------- | ----------------------------------------------------------- |
| `Domain`  | Target Active Directory domain (defaults to current domain) |

---

## 🌐 2. Test Domain Controller Connectivity

**Function:** `Test-DCConnectivity`

Performs a connectivity test against one or more Domain Controllers using `Test-NetConnection`.

### Parameter

| Parameter           | Description                         |
| ------------------- | ----------------------------------- |
| `DomainControllers` | List of Domain Controller hostnames |

---

## 📑 3. Retrieve Active Directory Event Log Errors

**Function:** `Get-ADEventLogErrors`

Retrieves recent Directory Service errors and warnings from the Windows Event Log.

### Parameter

| Parameter   | Description                             |
| ----------- | --------------------------------------- |
| `LastHours` | Number of hours to search (default: 24) |

---

# 🚀 Usage Examples

### Generate an Active Directory Health Report

```powershell
Get-ADDomainHealthStatus
```

### Test Domain Controller Connectivity

```powershell
Test-DCConnectivity -DomainControllers "DC1","DC2","DC3"
```

### View Recent Directory Service Errors

```powershell
Get-ADEventLogErrors -LastHours 24
```
