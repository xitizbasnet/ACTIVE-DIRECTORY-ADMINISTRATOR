# 📄 Script Code

```powershell id="yhgtkm"
# L1.3: DNS Records Query Script
# ================================

# QUERY DNS FOR AD SRV RECORDS
function Get-ADDNSSrvRecords {
    param([string]$Domain)

    Write-Host "=== AD SRV Records for $Domain ===" -ForegroundColor Cyan

    $SrvRecords = @(
        "_ldap._tcp.dc._msdcs.$Domain",
        "_ldap._tcp.$Domain",
        "_kerberos._tcp.dc._msdcs.$Domain",
        "_kerberos._tcp.$Domain",
        "_gc._tcp.$Domain"
    )

    foreach ($Record in $SrvRecords) {
        try {
            $Resolved = [System.Net.Dns]::GetHostAddresses($Record)
            Write-Host "3 $Record : $($Resolved.IPAddressToString)"
        }
        catch {
            Write-Host "7 $Record : NOT FOUND"
        }
    }
}

# VERIFY A RECORDS
function Test-DNSARecords {
    param([string[]]$Hostnames)

    Write-Host "`n=== DNS A Record Verification ===" -ForegroundColor Cyan

    foreach ($Host in $Hostnames) {
        try {
            $IPs = [System.Net.Dns]::GetHostAddresses($Host)

            foreach ($IP in $IPs) {
                Write-Host "3 $Host resolves to $IP"
            }
        }
        catch {
            Write-Host "7 Cannot resolve $Host"
        }
    }
}

# REVERSE DNS LOOKUP
function Test-ReverseDNS {
    param([string[]]$IPAddresses)

    Write-Host "`n=== Reverse DNS Lookup ===" -ForegroundColor Cyan

    foreach ($IP in $IPAddresses) {
        try {
            $HostName = [System.Net.Dns]::GetHostByAddress($IP).HostName
            Write-Host "3 $IP -> $HostName"
        }
        catch {
            Write-Host "7 No reverse record for $IP"
        }
    }
}

# ===== USAGE EXAMPLES =====

# Test-ReverseDNS -IPAddresses "192.168.1.10","192.168.1.11"
```

# ⚙️ Functions Included

## 🌐 1. Query Active Directory SRV Records

**Function:** `Get-ADDNSSrvRecords`

Retrieves and verifies the DNS Service (SRV) records required for Active Directory functionality.

### Parameter

| Parameter | Description                  |
| --------- | ---------------------------- |
| `Domain`  | Active Directory domain name |

### SRV Records Checked

* `_ldap._tcp.dc._msdcs.<domain>`
* `_ldap._tcp.<domain>`
* `_kerberos._tcp.dc._msdcs.<domain>`
* `_kerberos._tcp.<domain>`
* `_gc._tcp.<domain>`

---

## 🖥️ 2. Verify DNS A Records

**Function:** `Test-DNSARecords`

Resolves one or more hostnames to their corresponding IP addresses.

### Parameter

| Parameter   | Description                         |
| ----------- | ----------------------------------- |
| `Hostnames` | One or more DNS hostnames to verify |

---

## 🔄 3. Perform Reverse DNS Lookup

**Function:** `Test-ReverseDNS`

Performs reverse DNS lookups to verify that IP addresses correctly resolve to hostnames.

### Parameter

| Parameter     | Description                       |
| ------------- | --------------------------------- |
| `IPAddresses` | One or more IP addresses to query |

---

# 📊 Verification Checks

This script verifies the following DNS components:

* 🌐 Active Directory SRV records
* 🖥️ DNS A (Host) records
* 🔄 Reverse DNS (PTR) records
* 📍 Hostname-to-IP resolution
* 📍 IP-to-Hostname resolution
