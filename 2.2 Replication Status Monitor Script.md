# 📘 2.2 Replication Status Monitor Script

## 📋 Overview

**Purpose:** Monitor Active Directory replication health across all Domain Controllers and identify replication or connectivity issues.

| Property      | Value                                                       |
| ------------- | ----------------------------------------------------------- |
| 🎯 Purpose    | Monitor AD replication health across all domain controllers |
| 📈 Difficulty | Moderate                                                    |
| 💼 Use Case   | Proactive monitoring                                        |

---

# 📄 Script Code

```powershell
# L2.2: Replication Status Monitor Script
# ==========================================

function Get-ADReplicationHealth {
    param([string]$Domain = (Get-ADDomain).DNSRoot)

    Write-Host "=== AD Replication Health Report ===" -ForegroundColor Cyan

    $DomainControllers = Get-ADDomainController -Filter * -Server $Domain
    $HealthIssues = @()

    foreach ($DC in $DomainControllers) {

        Write-Host "`nChecking $($DC.Name)..." -ForegroundColor Yellow

        try {

            # Test replication
            $ReplStatus = Get-ADReplicationStatus -EnumerationServer $DC.HostName

            foreach ($Repl in $ReplStatus) {

                $LastSuccessTime = $Repl.LastReplicationSuccess
                $TimeSinceReplication = (Get-Date) - $LastSuccessTime

                if ($TimeSinceReplication.TotalMinutes -gt 60) {

                    Write-Host " 7 ISSUE: $($Repl.Partner) - " `
                        "Last replication: $($TimeSinceReplication.TotalMinutes) min ago"

                    $HealthIssues += $Repl
                }
                else {

                    Write-Host " 3 OK: $($Repl.Partner) - " `
                        "Last replication: $([int]$TimeSinceReplication.TotalMinutes) min ago"
                }
            }
        }
        catch {

            Write-Host " 7 Error connecting to $($DC.Name): $_"

            $HealthIssues += $_
        }
    }

    # Summary
    Write-Host "`n=== SUMMARY ===" -ForegroundColor Cyan

    if ($HealthIssues.Count -eq 0) {

        Write-Host "3 All replications healthy" -ForegroundColor Green
    }
    else {

        Write-Host "7 $($HealthIssues.Count) replication issues detected" -ForegroundColor Red
    }

    return $HealthIssues
}

function Test-DCConnectivityAdvanced {
    param([string[]]$DomainControllers)

    Write-Host "`n=== Advanced DC Connectivity Test ===" -ForegroundColor Cyan

    $Ports = @(53, 88, 135, 389, 445, 3268, 3389)

    foreach ($DC in $DomainControllers) {

        Write-Host "`n$DC:" -ForegroundColor Yellow

        foreach ($Port in $Ports) {

            $Test = Test-NetConnection -ComputerName $DC -Port $Port `
                -WarningAction SilentlyContinue

            $Status = if ($Test.TcpTestSucceeded) { "3" } else { "7" }

            Write-Host " Port $Port : $Status"
        }
    }
}

function Invoke-ReplicationSync {
    param([string]$SourceDC, [string]$DestinationDC)

    Write-Host "Syncing replication: $SourceDC -> $DestinationDC..." `
        -ForegroundColor Cyan

    try {

        Get-ADReplicationStatus -EnumerationServer $SourceDC |
            Where-Object { $_.PartnerName -like "*$DestinationDC*" } |
            ForEach-Object { Sync-ADObject -Object $_.PartnerName }

        Write-Host "3 Replication sync initiated" -ForegroundColor Green
    }
    catch {

        Write-Host "7 Error: $_" -ForegroundColor Red
    }
}
```

# ⚙️ Functions Included

## 🔄 1. Monitor Active Directory Replication Health

**Function:** `Get-ADReplicationHealth`

Checks the replication status of every Domain Controller in the specified domain and reports any replication delays or failures.

### Parameter

| Parameter | Description                                                     |
| --------- | --------------------------------------------------------------- |
| `Domain`  | Target Active Directory domain (defaults to the current domain) |

### Health Checks Performed

* 🔄 Replication status
* ⏱️ Last successful replication timestamp
* ⚠️ Replication delays exceeding 60 minutes
* 📊 Overall replication summary

---

## 🌐 2. Advanced Domain Controller Connectivity Test

**Function:** `Test-DCConnectivityAdvanced`

Tests TCP connectivity to essential Active Directory service ports on one or more Domain Controllers.

### Parameter

| Parameter           | Description                             |
| ------------------- | --------------------------------------- |
| `DomainControllers` | One or more Domain Controller hostnames |

### Ports Tested

| Port | Service              |
| ---- | -------------------- |
| 53   | DNS                  |
| 88   | Kerberos             |
| 135  | RPC Endpoint Mapper  |
| 389  | LDAP                 |
| 445  | SMB                  |
| 3268 | Global Catalog       |
| 3389 | Remote Desktop (RDP) |

---

## 🔁 3. Initiate Replication Synchronization

**Function:** `Invoke-ReplicationSync`

Attempts to initiate replication between a source and destination Domain Controller.

### Parameters

| Parameter       | Description                   |
| --------------- | ----------------------------- |
| `SourceDC`      | Source Domain Controller      |
| `DestinationDC` | Destination Domain Controller |

---
