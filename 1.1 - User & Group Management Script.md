# 📘 1.1 User & Group Management Script

## 📋 Overview

**Purpose:** Create users, add users to groups, disable accounts, and manage basic Active Directory user properties.

| Property      | Value                                                                            |
| ------------- | -------------------------------------------------------------------------------- |
| 🎯 Purpose    | Create users, add users to groups, disable accounts, and manage basic properties |
| 📈 Difficulty | Easy                                                                             |
| 💼 Use Case   | Daily user administration                                                        |

---

# 📄 Script Code

```powershell
# L1.1: User & Group Management Script
# ==========================================

# 1. CREATE NEW USER
function New-ADUserBatch {
    param(
        [string]$FirstName,
        [string]$LastName,
        [string]$OUPath = "OU=Users,DC=yourdomain,DC=com"
    )

    $SamAccountName = ($FirstName.Substring(0,1) + $LastName).ToLower()
    $UserPrincipalName = "$SamAccountName@yourdomain.com"
    $Password = ConvertTo-SecureString "TempPass@2024" -AsPlainText -Force

    try {
        New-ADUser -Name "$FirstName $LastName" `
            -SamAccountName $SamAccountName `
            -UserPrincipalName $UserPrincipalName `
            -Path $OUPath `
            -AccountPassword $Password `
            -Enabled $true `
            -ChangePasswordAtLogon $true `
            -ErrorAction Stop

        Write-Host "3 User created: $SamAccountName" -ForegroundColor Green
    }
    catch {
        Write-Host "7 Error: $_" -ForegroundColor Red
    }
}

# 2. ADD USER TO GROUP
function Add-UserToGroup {
    param(
        [string]$UserName,
        [string[]]$GroupNames
    )

    foreach ($Group in $GroupNames) {
        try {
            Add-ADGroupMember -Identity $Group -Members $UserName -ErrorAction Stop
            Write-Host "3 Added $UserName to $Group" -ForegroundColor Green
        }
        catch {
            Write-Host "7 Failed: $_" -ForegroundColor Red
        }
    }
}

# 3. DISABLE USER ACCOUNT
function Disable-ADUserAccount {
    param([string]$UserName)

    try {
        Disable-ADAccount -Identity $UserName -ErrorAction Stop
        Write-Host "3 User $UserName disabled" -ForegroundColor Green
    }
    catch {
        Write-Host "7 Error: $_" -ForegroundColor Red
    }
}

# 4. LIST ALL USERS IN OU
function Get-OUUsers {
    param([string]$OUPath)

    Get-ADUser -Filter * -SearchBase $OUPath |
        Select-Object Name, SamAccountName, Enabled, LastLogonDate |
        Format-Table -AutoSize
}

# ===== USAGE EXAMPLES =====

# New-ADUserBatch -FirstName "John" -LastName "Doe"

# Add-UserToGroup -UserName "jdoe" -GroupNames "IT-Staff","VPN-Users"

# Disable-ADUserAccount -UserName "jdoe"

# Get-OUUsers -OUPath "OU=Users,DC=yourdomain,DC=com"
```

# ⚙️ Functions Included

## 👤 1. Create New User

**Function:** `New-ADUserBatch`

Creates a new Active Directory user account using the supplied first name, last name, and Organizational Unit (OU).

### Parameters

| Parameter   | Description                                                           |
| ----------- | --------------------------------------------------------------------- |
| `FirstName` | User's first name                                                     |
| `LastName`  | User's last name                                                      |
| `OUPath`    | Target Organizational Unit (default: `OU=Users,DC=yourdomain,DC=com`) |

---

## 👥 2. Add User to Group

**Function:** `Add-UserToGroup`

Adds an existing Active Directory user to one or more security groups.

### Parameters

| Parameter    | Description                |
| ------------ | -------------------------- |
| `UserName`   | Active Directory username  |
| `GroupNames` | One or more AD group names |

---

## 🔒 3. Disable User Account

**Function:** `Disable-ADUserAccount`

Disables an existing Active Directory user account.

### Parameters

| Parameter  | Description               |
| ---------- | ------------------------- |
| `UserName` | Active Directory username |

---

## 📄 4. List All Users in an OU

**Function:** `Get-OUUsers`

Retrieves all users from the specified Organizational Unit and displays:

* Name
* SamAccountName
* Enabled Status
* Last Logon Date

### Parameters

| Parameter | Description                   |
| --------- | ----------------------------- |
| `OUPath`  | Organizational Unit to search |

---
